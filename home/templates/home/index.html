Wifi Sharing

About

Wifi sharing (not actual title) is an application developed for MHacks 2014 as a Mac exclusive (sorry) program designed to heatmap wifi signal strengths in a location. This data is crowd sourced, so it is constantly being updated and refined to create the most accurate heatmap possible. Using this application, it is possible to go to a new place and instantly know where to sit for the strongest wifi signal, because that is clearly very important.
